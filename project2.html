Overview

Me and my group have started a project, a project of which we've decided that it'll be a text adventure game with a story inspired by other medias/series. It's also based on the story that I'm currently writing.

End Users

The End Users that we've decided for this project are people who enjoy reading fictional stories such as medieval stories. They have to be 13+ as it may contain violent content.

Requirements

  • Structure/Layout
    • Must be easy to follow/read
    • Must be organised
  • Options
    • The user chooses what route they want to take.
  • Usage of certain language
    • Characters are made originally with original names. Original concepts and an inspired story.
  • Introduction
    • In the start of the game, the user will be provided instructions.
    • The user is introduced to the world.

Implications

  • Functionality
  • I addressed this by:
    • Ensuring that our code works as intended by testing it to make sure it works like how we intended it to be. Code must be double checked.
  • Intellectual Property
  • I addressed this by:
    • All characters that feature in the game are originally made by Lejan. The magic system in the game/story is not entitled to copyright laws but I will make it more unique. The storyline will also be unique.
  • Usability
  • I addressed this by:
    • Ensuring that the game is easy to use by providing instructions every now and then. When the user encounters a multi choice question, they are provided with choices. We'll also code the game in an extent for it to be efficient
