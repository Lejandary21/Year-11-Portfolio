Task 1: Text Adventure Game

Development

Me and my group have started a Text Adventure Game project in Digital Technology. My group is coding the scenarios in the game with different outcomes as it progresses.

The areaChoice is a variable needed for the code to work. When the user inputs his choice which is either the cave or forest. When the player chooses, the code will store the choice and brings them to its designated function, and should run as it should be.

When the player doesn't choose either of the two choices, the else code will run. The code will not proceed unless the user chooses either the cave or forest.
